What is it?

An educational mobile application to practice simple multiplication problems.


Note: this was before i knew what color schemes were

My first real intro to app development

I took AP Computer Science my senior year of high school and created this for my final project. I used Java and the IDE Android Studios.

But why build this?

Our final project was extremely open-ended. The only requirements were to demonstrate our understanding of the programming language Java and apply it to the real world.

As I was exploring ideas, I tried to think of problems around me that I could solve. I remembered my mother mentioning she was teaching multiplication problems to some of her more advanced students and decided I could help.

Technical challenges

I had no prior experience with android app development besides a course on Swift. Therefore, I initially had to get over a small learning curve for Android Studios.

Customization! Figuring out how to set the number of questions depending on the user input and then updating it all live in my app was something I had trouble with because it was something I had never really learned in class.

Retrieving the correct answer to display. If the user submitted an incorrect answer, I had to pull the correct answer from an array and display it.

Final Score! Redirecting the buttons! I had to keep track of the number of correct answers to display a final score. I then had an end screen that allowed them to play again or redirect them back to the home page. I needed to process the connections between each of my pages.

What I would do differently
  • Create a landing page before presenting the main page to welcome users
  • Implement constraints depending on the user's screen size
  • Display the correct answer regardless of the user gets the question correct or not
  • Create a scoreboard similar to Quizlet to compare with other students in the class
